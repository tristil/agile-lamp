Agile Lamp Driver

Get Version

0.1.0

What

A user-space driver for the Agile Lamp USB Lava Lamp

Installing

sudo gem install agilelamp-driver

The basics

After installing the gem, you should be able to control the Agile Lamp while it is plugged in:

  • agilelamp-driver red # red means bad!
  • agilelamp-driver green # green means good!
  • agilelamp-driver various # meaningless cycling colors
  • agilelamp-driver bell # the bell is very unpleasant

Source code

You can fetch the source from tarball or git repository:

  • rubyforge: http://rubyforge.org/scm/?group_id=6281
  • gitorious: website — git://gitorious.org/agile-lamp/mainline.git
git clone git://gitorious.org/agile-lamp/mainline.git

Build and test instructions

cd agilelamp-driver
rake test
rake install_gem
